# A.M.I.S.X.
Atari Message and Information System eXpanded

This application is intended to be run on actual Atari 8-bit computer hardware with the #FujiNet peripheral.

Outline:
  - Wait for call
  - Login
  - Signup
  - Message Base
  - Files 
  - Email
  - Print Text Files
  - Change between ASCII/ATASCII
  - External Applications (Doorgames)
  - Page SysOp
